He implemented a model for classification task. The model includes data sampling using SMOTE, XGBoost algorithm, and cross-validation. To increase the accuracy, the author also implemented hyperparameter tuning using randomized search. The best combination of hyperparameters was then used to train the final model. His repository on github.

His contribution involved using the pickle library to obtain the best trained model and creating a Flask application with complete frontend UI design. The Flask application was then deployed on Heroku to make it accessible for others to use. His repository on github.

Kamlesh K

He implemented 3 types of sampling techniques (under-sampling, over-sampling, and hybrid-sampling) to handle imbalanced datasets. He implemented 2 types of over-sampling (random over and renn), 2 types of under-sampling (random under and smote), and 4 combinations of hybrid-sampling. This resulted in 8 different datasets with equal counts of affected and unaffected rows.

Ajeeth

He implemented multiple classifiers for an imbalanced dataset and used stratified k-fold cross-validation for each model. He also calculated accuracy for each dataset and found that accuracy varies for each one. In cross-validation, he trained the training folds using the respective model and tested in testing fold to predict accuracy.
